# is-max
Determine if a value is lower than another value

## Install
```
$ npm install --save is-max
```

## Usage
```js
const isMax = require('is-max');

console.log(isMax('Hello World!', 20));
//=> true

console.log(isMax('Hello', 4));
//=> false
```
